# Phase Guide React App

Interactive Thousand Sons phase flow built with React + Vite.

## Scripts

```powershell
# install deps
npm install

# run dev server (default http://localhost:5173)
npm run dev

# create production build (outputs to dist/)
npm run build

# preview production build
npm run preview
```

Run all commands from `phase-guide-react`. Use `npm` or a compatible package manager (pnpm, yarn) if preferred.
